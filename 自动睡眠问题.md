# 概览

在Win10使用中遇到一个问题，但使用代理软件共享网络时，系统会自动休眠，即使在电源选项中设置了`使计算机进入睡眠状态`为`从不`，依然会遇到这个问题。

# 解决

这是因为Win10存在**无人参与系统睡眠超时**这个概念，优先级高于上述设置。而这个设置选项默认是隐藏的，需要修改注册表开启：

1. regedit.exe
2. `HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Power\PowerSettings\238C9FA8-0AAD-41ED-83F4-97BE242C8F20\7bc4a2f9-d8fc-4469-b07b-33eb785aaca0`
3. 修改`attributes`的值为2
4. 在电源选项中的`高级电源设置`会出现`无人参与系统睡眠超时`，修改其值即可。
